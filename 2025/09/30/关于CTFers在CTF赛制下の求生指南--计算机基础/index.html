关于CTFers在CTF赛制下の求生指南--计算机基础

debu8ger2025-09-302026-03-11

XAUTCTFers特供版—by SS-Team

  • Windows的键盘快捷键(常用)：

    1. Ctrl+C&Ctrl+V ：复制粘贴
    2. Ctrl+A&Ctrl+Z ：选择全部&撤销操作
    3. Win+E&Win+R ：打开文件资源管理器&运行框
    4. Ctrl+F ：查找字符串（对于一些简单の签到题查找flag字符串）
    5. Ctrl+N&Ctrl+S ：新建窗口等&保存（对于编程，尤其python每次run都得保存）
    6. Ctrl+点击多个文件&Ctrl+拖动文件 ：多选文件&复制文件
    7. Alt+Tab&Alt+Enter ：切换不同正在运行の程序&查看所选项属性（提高效率）

  • Linux的常用命令：

    1. ls /&ls -al ：显示目标列表(完整目录)&显示目录所有文件(隐藏文件–可能有用)
    2. cd [path]&pwd ：切换工作目录&显示当前工作目录的绝对路径
    3. nc ip port ：建立TCP/UDP连接或监听端口(pwn✌)
    4. cat /[file]&less [file] ：在终端打印文件内容(哈哈，cat /flag)&浏览文件内容
    5. mv [file1] [file2]&cp/mv [file1] [file2] ：移动/重命名文件&复制/删除文件
    6. grep [keyword] [file]&find [file] ：在文件中搜索指定字符串&在文件系统中搜索文件(与Ctrl+F差不多)
    7. mkdir/rmdir [dir]&touch [file] ：创建/删除目录&创建空文件
    8. chmod [options(777 / +x)] mode [file]&chown [options] owner[:group] [file] ：修改文件权限(便于附件调试，一般可root)&修改文件操作者
    9. ps aux&kill [PID] ：查看当前运行进程&杀死进程
    10. nano / vi / emacs [file] ：编辑文件
    11. 管道符 | ：连接命令
    12. echo [strings] (+ > [file])&strings [file] ：打印文本(可写入文件)&查找文件中所有字符串

  • 科学上网(梯子)：

    1. 众所周知，国内外的网络间有一堵无形的墙阻隔，它便是中国国家防火墙，简称GFW，这导致国内用户在访问一些境外网站时受阻，但有时我们所需要的资料工具只有在境外才能获取，对于计算机相关专业更是如此，因此学会“科学上网”是必要的，“梯子”便是绕过这一限制的工具
    2. 目前来说比较好的梯子是 Clash ,有很多机场(VPN),其中 ikuu VPN是免费的，但是选择性不稳定，显示超时，足够日常使用
    3. 这里不便于展开介绍，可以看看我们学校论坛上的**梯子机场专题推荐**

  • 搜索引擎&LLM的选择：

    1. 首先搜索引擎的话，尽量不要选择百度(国内の…一言难尽)，搜到的不全，广告多；如果有条件的话(科学上网)首选Google。当然，必应也是非常推荐的
    2. LLM的话，首选ChatGPT，当然也推荐国内的deepseek、通义千问、kimi，但像kimi这种有深度思考功能的如果算力不够(难题)容易自己陷入死循环中，不输出结果
    3. 当然，也欢迎本地部署AI和知识库
